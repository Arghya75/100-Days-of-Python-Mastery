Prime Number Checker
Overview:
The Prime Number Checker program is a simple Python script designed to determine whether a given number is a prime number. Prime numbers are natural numbers greater than 1 that have no positive divisors other than 1 and themselves. This program takes an input number and checks its divisibility by potential factors to determine if it is prime.

Use Cases:
- Math Enthusiasts : Quickly check if a number is prime for mathematical problems.
- Learning Tool : Ideal for those learning about prime numbers and basic programming.
