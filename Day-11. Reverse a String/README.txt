Reverse a String

Write a function that takes a string as input and returns the entire string reversed.

Example 1:
    - Input: `"hello"`
    - Output: `"olleh"`

Example 2:
    - Input: `"Hello World"`
    - Output: `"dlroW olleH"`

Example 3:
    - Input: `"Python is fun!"`
    - Output: `"!nuf si nohtyP"`
